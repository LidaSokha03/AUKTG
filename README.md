# AUKTG# Career Assistant Telegram Bot

## Description
Career Assistant is a Telegram bot that helps students and young professionals create a professional CV, prepare for job interviews, and store their career-related data.

## Main Functionality
- Dashboard - main menu with quick commands (/register, /login, /cv, /interview, /export, /import)  
- CV Builder - step-by-step form for entering user data with live preview and export to PDF or DOCX  
- Interview Simulator - 5-8 interview questions, timer, score, and summary report  
- Import / Export - ability to save or restore user profiles from a file  
- Erase Data - deletes all user data

## Technologies
Python 3.12  
Telegram API - pyTelegramBotAPI  
Database - SQLite with SQLAlchemy  
File Generation - fpdf (PDF), python-docx (DOCX)  
LLM (optional AI integration) - question generation and CV enrichment

## Bot Commands
/start - greeting and menu  
/form_pdf - generate CV in PDF format  
/form_docx - generate CV in DOCX format  
/register - register a new user  
/login - log in to existing profile  
/cv - open CV builder interface  
/interview - start interview simulation  
/export - export profile data  
/import - import saved data  
/erase_me - permanently delete account and data  
/lang - switch language (EN / UA)  

## Data and Privacy
The bot collects only essential user data: name, contact information, education, work experience, and skills.  
No passwords or sensitive data are stored.  
Users can permanently delete their data using /erase_me.
